# VangaTypePanalam Roadmap

This document outlines the planned features and milestones for the VangaTypePanalam project.

## 🚀 Phase 1: Core Foundation (Completed ✅)
- [x] **Adaptive Typing Engine**: Real-time per-key latency and accuracy tracking using circular buffers.
- [x] **Multi-Script Support**: English (QWERTY), Tamil (Tamil99), and Tanglish.
- [x] **30-Level Curriculum**: Progressive touch typing lessons for all supported languages.
- [x] **Offline-First Storage**: Full IndexedDB integration for zero-latency local data persistence.
- [x] **Virtual Keyboard**: Heatmap overlays, finger guides, and press feedback.
- [x] **Stats Dashboard**: Session history, daily activity heatmaps, and WPM trends.

## 🌟 Phase 2: Engagement & Social (Current 🔨)
- [x] **Gamification V2**: Dynamic Rank system, Badge collection, and XP progression.
- [x] **Cloud Sync**: Optional background synchronization to Neon PostgreSQL via Auth.js & Prisma.
- [x] **Admin Central**: Internal dashboard for managing gamification assets and SVG injection.
- [x] **Multiplayer Foundations**: Socket.IO integration for real-time typing races (ready for deployment).
- [ ] **Sound Expansion**: Diverse mechanical keyboard sound profiles (Blue, Brown, Red switches) and error tactile feedback.
- [ ] **Full PWA Support**: Service Worker integration via Serwist for true "installable" offline capability.
- [ ] **Internationalization (i18n)**: Multi-language UI support (Tamil/English interface toggle).

## 🏆 Phase 3: Advanced Features (Planned)
- [ ] **Bulk Cloud Restore**: Seamless one-click restoration of full local profile on new device login.
- [ ] **Multiplayer Public Lobby**: Global matchmaking and ranked leaderboards.
- [ ] **Team Relays**: Cooperative typing modes for groups.
- [ ] **Custom Lesson Creator**: Allow users/teachers to create and share their own custom typing curricula.
- [ ] **Advanced Analytics**: Digraph (letter pair) and trigraph heatmap analysis for pinpointing transition bottlenecks.
- [ ] **Mobile Touch Optimization**: Enhanced virtual keyboard for tablets and touch devices.

---

## 📈 Long-term Vision
To become the premier open-source typing tutor for Indic scripts, bridging the gap between phonetic input and standard keyboard layouts.
